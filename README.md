# 📊 Data Cleaning & Preparation Project

## 📌 Overview

This project was completed as part of the **DecodeLabs Data Analytics Internship**.

The objective was to clean and prepare a raw e-commerce dataset using Microsoft Excel by handling missing values, removing duplicate records, correcting data formats, and validating the dataset before analysis.

---

## 🎯 Objectives

- Handle missing values
- Remove duplicate records
- Standardize date, number, and text formats
- Improve data quality
- Prepare the dataset for future analysis

---

## 🛠 Tools Used

- Microsoft Excel 2016

---

## ✅ Data Cleaning Tasks

- ✔ Checked missing values
- ✔ Removed duplicate OrderIDs
- ✔ Standardized date format
- ✔ Corrected text formatting
- ✔ Removed extra spaces
- ✔ Checked spelling consistency
- ✔ Verified Quantity, UnitPrice, and TotalPrice
- ✔ Reviewed TrackingNumber consistency
